sudo apt install python3.13-venv

rm -rf venv

(probeer opnieuw)
python3 -m venv venv

source venv/bin/activate

Installeer de pip packages:

pip install -r requirements.txt
dit installeert alles wat mogelijk handig zou kunnen zijn

als niks werkt op linux, probeer de com porten om te wisselen. dit kan in Tag_V3.0/Common/sara_ports.py onder  

    COM_HEAD_LINUX: str = "/dev/ttyACM1"
    '''Standaard COM-poort voor het hoofd van de robot op Linux.'''

    COM_BODY_LINUX: str = "/dev/ttyACM0"
    '''Standaard COM-poort voor het lichaam van de robot op Linux.'''

hetzelfde geld voor windows ongeveer, behalve dat het ook kan zijn dat je je poorten niet op COM10/11 hebt staan. het is veel makkelijker om het in dit bestand aan te passen, dus dat kan je ook proberen.

Voor de camera, moet je iets gebruiken genaamd astraSDK. dit zorgt voor de communicatie met de camera. de setup hiervan is nogal priegelwerk, maar uiteindelijk werkt het wel. PS: hier klopt geen ene bal van. volg deze instructie niet.

Hieronder de instructie voor meeste standaard linux distros:

sudo apt update

sudo apt install cmake build-essential g++

open SaraJvB-main/OpenCV/CameraUbuntu/AstraSDK-v2.1.3-Ubuntu-x86_64

unzip een van de 2, ik heb 18.04 gekozen want dit is de laatste versie.

verplaats de folder naar CameraUbuntu, en noem hem iets anders, zoals "AstraSDK-v2.1.3"

cd ~/saraJvB-main/OpenCV/CameraUbuntu/AstraSDK-v2.1.3/samples

mkdir build && cd build

cmake ..

make -j$(nproc)
